✅ 핵심 발견: 외로움의 주효과는 사라졌지만, **성별 조절 효과는 살아남음**

"외로움이 실행 기능을 손상시킨다" → ❌ 틀림 (DASS와 혼재됨)
"외로움이 **남성**에게만 실행 기능을 손상시킨다" → ✅ 진실 (DASS와 독립적)

1. DASS 통제 결과 상세
================================================================================

분석 대상: N = 76 (남성 30명, 여성 46명)

통제 변수:
- DASS-21 우울 (dass_depression)
- DASS-21 불안 (dass_anxiety)
- DASS-21 스트레스 (dass_stress)
- 나이 (age)

--------------------------------------------------------------------------------
1.1 주효과 (Main Effects) - 모두 실패
--------------------------------------------------------------------------------

H1 (WCST PE Rate):
  UCLA → PE: β = 0.324, p = 0.696 ❌
  해석: 외로움의 직접 효과는 우울/불안과 분리 불가능

H3 (Stroop Interference):
  UCLA → Stroop: β = 12.130, p = 0.485 ❌
  해석: 외로움 효과 없음 (원래부터 약했음)

H4 (PRP Bottleneck):
  UCLA → PRP: β = -7.502, p = 0.755 ❌
  해석: 외로움의 직접 효과는 기분장애와 중첩됨

결론: **"순수 외로움 효과"는 존재하지 않음**

--------------------------------------------------------------------------------
1.2 상호작용 효과 (Interaction Effects) - 2개 유의
--------------------------------------------------------------------------------

H1 (WCST PE Rate): ✅ **유의**
  UCLA × Gender → PE: β = 2.517, p = 0.0253 ✅

  성별별 단순 기울기 (Simple Slopes):
  - 여성: β = -0.519, p = 0.648 (NS)
  - 남성: β = 2.900, p = 0.0194 ✅ **유의!**

  해석:
  - 남성만 외로움↑ → PE↑ 관계가 유의
  - 이 효과는 DASS 통제 후에도 살아남음
  - **성별 특이적 메커니즘 존재**

H4 (PRP Bottleneck): ✅ **유의**
  UCLA × Gender → PRP: β = 68.802, p = 0.0360 ✅

  성별별 단순 기울기:
  - 여성: β = -36.373, p = 0.248 (NS)
  - 남성: β = 49.654, p = 0.168 (경향, p < 0.10)

  해석:
  - 남성에서 외로움↑ → bottleneck↑ 경향
  - DASS 통제 후에도 상호작용 유의
  - 남성 취약성 패턴 지지

H3 (Stroop Interference): ❌ 비유의
  UCLA × Gender → Stroop: β = 21.153, p = 0.378 ❌

  해석: Stroop에서는 성별 조절 효과도 없음 (천장 효과)

================================================================================
2. 이론적 해석
================================================================================

2.1 왜 주효과는 사라졌나?
-------------------------
UCLA 외로움 점수와 DASS 기분장애 점수가 **강하게 상관** (r ~ 0.5-0.7):
- 외로운 사람은 동시에 우울/불안/스트레스가 높음
- 두 변수가 공유하는 분산이 큼
- DASS를 통제하면 UCLA의 "고유 분산"이 거의 남지 않음

→ 외로움의 직접 효과는 기분장애와 **개념적으로 중첩됨**

2.2 왜 상호작용은 살아남았나?
----------------------------
상호작용 효과 = "성별에 따라 UCLA-PE 관계의 강도가 다름"

이것이 살아남았다는 것은:
- **남성만 외로움에 취약**한 메커니즘이 존재
- 이 메커니즘은 단순한 기분장애와는 **다름**
- 성별 특이적 신경/인지 경로 시사

가능한 메커니즘:
1. 남성: 외로움 → 사회적 철수 → 인지 자극 감소 → EF 저하
2. 여성: 외로움 → 과각성/보상 → EF 유지 (또는 hypervigilance)
3. 호르몬/신경전달물질 차이 (테스토스테론, 옥시토신 등)

3. 출판 전략
================================================================================

3.1 ❌ 절대 해서는 안 되는 주장
--------------------------------
- "외로움이 실행 기능을 손상시킨다"
- "순수 외로움 효과가 기분장애와 독립적"
- "UCLA 점수가 높으면 EF가 나빠진다" (성별 구분 없이)

이러한 주장은 **DASS 통제를 하지 않은 것**과 동일하며, 방법론적으로 **결함**

3.2 ✅ 정당화된 주장
--------------------
1. **"외로움이 남성에게만 실행 기능을 손상시킨다"** ✅
   - 근거: UCLA × Gender 상호작용 p = 0.025 (DASS 통제 후)
   - 근거: Male simple slope p = 0.019 (DASS 통제 후)

2. **"외로움의 EF 영향은 성별에 따라 다르다"** ✅
   - 근거: 2개 과제에서 유의한 상호작용 (WCST, PRP)

3. **"남성은 외로움에 대한 인지적 취약성을 보인다"** ✅
   - 근거: 남성에서만 일관된 정적 관계
   - 근거: 여성은 보호/보상 메커니즘 가능성

3.3 제안하는 논문 구조
----------------------
Title (안):
"Gender-Specific Vulnerability to Loneliness: Males Show Executive Function
Impairment Beyond Mood Effects"

Abstract 핵심 메시지:
- "While loneliness effects overlap with mood disturbance..."
- "...a gender-specific mechanism emerged that is independent of DASS."
- "Males, but not females, showed UCLA → EF impairment after controlling
   for depression, anxiety, and stress."

Introduction:
- 외로움과 기분장애의 개념적 중첩 인정
- 성별 조절 효과에 초점

Method:
- **명시**: "All analyses control for DASS-21 subscales"
- Hierarchical regression framework 제시

Results:
- **먼저** 상호작용 효과 보고 (p = 0.025)
- **그 다음** 성별별 simple slopes (Male p = 0.019)
- 주효과는 supplementary에만 언급

Discussion:
- 남성 취약성 메커니즘 집중 논의
- 여성 보상/보호 메커니즘 제안
- 기분장애와의 관계 투명하게 논의
