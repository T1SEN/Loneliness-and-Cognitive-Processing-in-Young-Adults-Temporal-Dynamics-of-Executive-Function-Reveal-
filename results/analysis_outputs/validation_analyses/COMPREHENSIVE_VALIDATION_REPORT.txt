EXECUTIVE SUMMARY
================================================================================

This report presents three complementary validation analyses to assess the
robustness and reliability of the observed UCLA × Gender interaction effects:

1. Cross-Validation (CV): Does the interaction improve out-of-sample prediction?
2. Robust/Quantile Regression: Is the effect robust to outliers? Does it vary
   across the performance distribution?
3. Type M/S Error Simulation: What is the risk of effect size exaggeration?

KEY FINDINGS:
-------------
✓ WCST PE (primary outcome, p=0.025):
  - CV: No significant predictive improvement (ΔRMSE p=0.52)
  - Robust: Moderate outlier influence (-20% change)
  - Quantile: Effect strongest at τ=0.90 (vulnerable tail, β=5.34)
  - Type M error: 1.66× (66% exaggeration risk, MODERATE)
  - Type S error: 0% (sign error negligible)
  - Power: 34.7% (underpowered)

⚠ Ex-Gaussian tau (p=0.49, non-significant):
  - CV: No predictive improvement (ΔRMSE p=0.63)
  - Robust: HIGH outlier influence (+79% change)
  - Quantile: Highly variable across distribution
  - Type M error: 4.90× (EXTREME exaggeration risk)
  - Type S error: 9.6% (moderate sign error risk)
  - Power: 8.3% (severely underpowered)

✓ PRP bottleneck (p=0.036):
  - CV: No predictive improvement (ΔRMSE p=0.56)
  - Robust: Moderate outlier influence (+20% change)
  - Quantile: Effect strongest at lower quantiles (resilient group)
  - Type M error: 1.74× (74% exaggeration risk, MODERATE)
  - Type S error: 0% (sign error negligible)
  - Power: 30.6% (underpowered)

⚠ Stroop interference (p=0.38, non-significant):
  - CV: No predictive improvement (ΔRMSE p=0.19)
  - Robust: Very robust to outliers (-1% change)
  - Quantile: Effect strongest at τ=0.90 (vulnerable tail, β=77.49)
  - Type M error: 3.79× (EXTREME exaggeration risk)
  - Type S error: 6.9% (moderate sign error risk)
  - Power: 8.7% (severely underpowered)

OVERALL INTERPRETATION:
-----------------------
1. Statistical Significance ≠ Predictive Utility:
   - Despite in-sample significance for WCST PE and PRP, the UCLA × Gender
     interaction does NOT significantly improve out-of-sample prediction.
   - This suggests the interaction may be overfitted to the current sample.

2. Outlier Sensitivity:
   - WCST PE and PRP show moderate outlier influence (20% coefficient change).
   - Ex-Gaussian tau shows HIGH outlier influence (79% change).
   - Stroop is robust to outliers (1% change).

3. Distributional Heterogeneity ("Vulnerable Tail" Hypothesis):
   - WCST PE: Effect 3× stronger at τ=0.90 (5.34) vs median (1.54).
   - Stroop: Effect only emerges at high quantiles (τ=0.90: 77.49).
   - Interpretation: Loneliness effects concentrate among those already
     struggling (high PE, slow RT), not uniformly across population.

4. Effect Size Exaggeration Risk:
   - WCST PE and PRP: Moderate exaggeration (1.6-1.7× when significant).
   - Tau and Stroop: EXTREME exaggeration (3.8-4.9× when significant).
   - Implication: Observed effect sizes likely overestimate true effects,
     especially for outcomes with low original p-values.

5. Statistical Power Crisis:
   - All outcomes underpowered (power < 50%).
   - WCST PE: 34.7% power (should be >80%).
   - Low power directly increases Type M/S error rates.

ANALYSIS 1: CROSS-VALIDATION (Predictive Performance)
================================================================================

Method:
-------
5-fold cross-validation comparing:
- Base model: y ~ age + gender + DASS_dep + DASS_anx + DASS_str
- Full model: Base + UCLA + UCLA × Gender

Metrics: RMSE (Root Mean Squared Error), R², MAE (Mean Absolute Error)
Paired t-test on ΔMetric (Full - Base) across folds.

Results:
--------

1. WCST Perseverative Error Rate
   Base model:  RMSE = 4.53 ± 1.27, R² = -0.16 ± 0.17, MAE = 3.30 ± 0.67
   Full model:  RMSE = 4.46 ± 1.13, R² = -0.15 ± 0.25, MAE = 3.26 ± 0.47
   Δ (Full - Base):
     ΔRMSE = -0.07 ± 0.23, t = -0.71, p = 0.52 (ns)
     ΔR²   = +0.01 ± 0.14, t = +0.14, p = 0.89 (ns)
     ΔMAE  = -0.04 ± 0.29, t = -0.31, p = 0.77 (ns)

   Interpretation: No significant predictive improvement despite in-sample
   p=0.025. The interaction term does not generalize to new data.

2. Ex-Gaussian Tau
   Base model:  RMSE = 116.3 ± 13.3, R² = -0.07 ± 0.09, MAE = 93.5 ± 17.5
   Full model:  RMSE = 117.9 ± 13.5, R² = -0.10 ± 0.10, MAE = 90.4 ± 16.6
   Δ (Full - Base):
     ΔRMSE = +1.63 ± 6.90, t = +0.53, p = 0.63 (ns)
     ΔR²   = -0.03 ± 0.12, t = -0.58, p = 0.60 (ns)
     ΔMAE  = -3.05 ± 2.72, t = -2.51, p = 0.07 (marginal trend)

   Interpretation: No improvement. If anything, Full model slightly worse
   (positive ΔRMSE, negative ΔR²). Marginal MAE improvement not robust.

3. PRP Bottleneck Effect
   Base model:  RMSE = 140.3 ± 21.3, R² = -0.15 ± 0.34, MAE = 114.5 ± 19.2
   Full model:  RMSE = 142.5 ± 19.0, R² = -0.19 ± 0.35, MAE = 114.4 ± 14.2
   Δ (Full - Base):
     ΔRMSE = +2.17 ± 7.53, t = +0.64, p = 0.56 (ns)
     ΔR²   = -0.04 ± 0.14, t = -0.61, p = 0.57 (ns)
     ΔMAE  = -0.03 ± 6.43, t = -0.01, p = 0.99 (ns)

   Interpretation: Despite in-sample p=0.036, no predictive benefit.

4. Stroop Interference
   Base model:  RMSE = 97.2 ± 23.6, R² = -0.21 ± 0.30, MAE = 76.2 ± 16.0
   Full model:  RMSE = 101.9 ± 22.1, R² = -0.38 ± 0.55, MAE = 79.3 ± 15.3
   Δ (Full - Base):
     ΔRMSE = +4.67 ± 6.64, t = +1.57, p = 0.19 (ns)
     ΔR²   = -0.17 ± 0.26, t = -1.48, p = 0.21 (ns)
     ΔMAE  = +3.05 ± 6.93, t = +0.98, p = 0.38 (ns)

   Interpretation: No improvement. Actually trending toward worse prediction.

Cross-Validation Conclusion:
----------------------------
✗ CRITICAL: The UCLA × Gender interaction does NOT provide significant
  out-of-sample predictive utility for ANY of the four outcomes.

Implications:
- In-sample statistical significance (p < 0.05) does not guarantee the effect
  generalizes to new participants.
- The interaction may be capturing sample-specific noise rather than a robust
  population-level relationship.
- For publication: Report both in-sample p-values AND cross-validated
  predictive metrics. The latter provides stronger evidence of replicability.

Part B: Quantile Regression (Distributional Heterogeneity)
-----------------------------------------------------------

Method: Test if UCLA × Gender effect varies across outcome distribution.
τ = 0.25 (lower quartile), 0.50 (median), 0.75, 0.90 (top decile).

"Vulnerable Tail" hypothesis: Effect strongest at high quantiles (poor performers).

Results (UCLA × Gender interaction coefficient β):

   τ = 0.25:  β = 2.48
   τ = 0.50:  β = 1.54
   τ = 0.75:  β = 1.58
   τ = 0.90:  β = 5.34  ← 3× stronger than median!
   Range: 3.80

   ✓✓ VULNERABLE TAIL CONFIRMED:
   Effect is 3× stronger among participants with highest PE rates (τ=0.90).
   Loneliness × male gender primarily harms those already struggling with
   cognitive flexibility, not the average male participant.

2. Ex-Gaussian Tau
   τ = 0.25:  β = 2.57
   τ = 0.50:  β = 56.53
   τ = 0.75:  β = 53.68
   τ = 0.90:  β = -48.47  ← Sign reversal!
   Range: 105.00

   ⚠ UNSTABLE: Sign reversal at τ=0.90 indicates extreme instability.
   Effect not consistent across distribution. Combined with 79% Huber change,
   this outcome is unreliable for inference.

3. PRP Bottleneck Effect
   τ = 0.25:  β = 107.35  ← Strongest at low quantile
   τ = 0.50:  β = 93.28
   τ = 0.75:  β = 100.09
   τ = 0.90:  β = -2.64   ← Nearly zero at high quantile
   Range: 109.98

   ✓ RESILIENT FLOOR PATTERN:
   Effect strongest among participants with SHORT bottleneck (good dual-task
   coordination). Opposite of vulnerable tail. Loneliness may disrupt optimal
   coordination more than already-impaired coordination.

4. Stroop Interference
   τ = 0.25:  β = -7.46   (small negative)
   τ = 0.50:  β = 14.79
   τ = 0.75:  β = 35.60
   τ = 0.90:  β = 77.49   ← 10× stronger than lower quartile!
   Range: 84.95

   ✓✓ VULNERABLE TAIL CONFIRMED:
   Effect only emerges at high quantiles. At τ=0.25 (fast responders),
   effect is near zero or negative. At τ=0.90 (slowest responders),
   effect is massive (77ms). Loneliness × male gender selectively impairs
   those already showing high interference.

Robust/Quantile Regression Conclusion:
---------------------------------------
1. Outlier Sensitivity:
   - WCST PE and PRP: MODERATE (20% change) - interpretable with caution.
   - Tau: EXTREME (79% change) - unreliable, do not interpret.
   - Stroop: Robust (1% change) - stable estimate.

2. Distributional Heterogeneity:
   ✓✓ MAJOR FINDING: Effects are NOT uniform across performance levels.

   WCST PE and Stroop: "Vulnerable tail" pattern

   → Average-performing males not affected.
   → Clinical implication: Lonely males with pre-existing EF deficits are

3. Implication for Mean-Level Analyses:
   ⚠ OLS regression (mean-level effect) MASKS important heterogeneity.

ANALYSIS 3: TYPE M/S ERROR SIMULATION (Effect Size Exaggeration Risk)
================================================================================

Method:
-------
Gelman & Carlin (2014) framework for quantifying errors in underpowered studies:

1. Set conservative true effect: β_true = 70% × β_observed
2. Simulate 1000 datasets (N=76) with true effect + residual noise
3. Fit regression model on each simulated dataset
4. Calculate:
   - Power: P(p < 0.05)
   - Type M error: E[|β_hat| / |β_true| | significant]
     (exaggeration ratio for significant results)
   - Type S error: P(sign(β_hat) ≠ sign(β_true) | significant)
     (probability of wrong sign when significant)

Interpretation Guide:
- Type M < 1.2: Minimal exaggeration (<20%)
- Type M 1.2-2.0: Moderate exaggeration (20-100%)
- Type M > 2.0: High exaggeration (>2×)

Results:
--------

1. WCST Perseverative Error Rate (Primary Outcome)
   Observed β = 2.52, p = 0.025
   True β (conservative) = 1.76

   Power: 34.7% (should be >80% for reliable inference)
   Type M error: 1.66× (66% exaggeration)
   Type S error: 0.0% (no sign errors)

   Interpretation:
   - MODERATE exaggeration risk.
   - When we get p<0.05 (only 35% of the time), our estimate is 1.66× the
     true effect on average.
   - If true β=1.76, our significant estimates average β_hat=2.93.
   - No risk of wrong sign (all positive when significant).
   - ⚠ Underpowered (35%) → increases exaggeration risk.

   Publication Implication:
   Report: "β=2.52 (95% CI: 0.32, 4.71), but power analysis suggests
   potential 66% exaggeration (Type M=1.66) if true effect is smaller."

2. Ex-Gaussian Tau (Non-Significant)
   Observed β = 19.24, p = 0.49
   True β (conservative) = 13.47

   Power: 8.3% (severely underpowered)
   Type M error: 4.90× (490% exaggeration!)
   Type S error: 9.6% (1 in 10 significant results have wrong sign)

   ⚠⚠ CRITICAL:
   - EXTREME exaggeration risk (5× true effect).
   - When this effect happens to reach p<0.05 (rare, 8%), it's almost
     certainly a false positive or massive overestimate.
   - 10% of significant results would have WRONG SIGN.
   - Combined with 79% Huber outlier influence: DO NOT INTERPRET THIS EFFECT.

   Publication Recommendation:
   Report as null (p=0.49) and do not speculate about "trends." Any future
   significant finding for tau needs independent replication with N>200.

3. PRP Bottleneck Effect
   Observed β = 68.80, p = 0.036
   True β (conservative) = 48.16

   Power: 30.6% (underpowered)
   Type M error: 1.74× (74% exaggeration)
   Type S error: 0.0% (no sign errors)

   Interpretation:
   - MODERATE exaggeration risk.
   - Significant estimates average 1.74× true effect.
   - If true β=48.16, we observe β_hat=83.59 on average when significant.
   - No sign error risk.
   - ⚠ Low power (31%) contributes to exaggeration.

   Publication Implication:
   Report p=0.036 but note moderate exaggeration risk (Type M=1.74).

4. Stroop Interference (Non-Significant)
   Observed β = 21.15, p = 0.38
   True β (conservative) = 14.81

   Power: 8.7% (severely underpowered)
   Type M error: 3.79× (379% exaggeration!)
   Type S error: 6.9% (1 in 15 significant results have wrong sign)

   ⚠⚠ CRITICAL:
   - EXTREME exaggeration risk (nearly 4× true effect).
   - When significant (rare, 9%), estimate is almost certainly inflated.
   - Despite robust Huber estimate (1% change), low power makes inference
     unreliable.
   - 7% sign error risk nontrivial.

   Publication Recommendation:
   Report as null (p=0.38). Do not over-interpret quantile regression
   finding (τ=0.90: β=77.49) without acknowledging it could be 4× exaggerated.

Type M/S Error Conclusion:
--------------------------

POWER CRISIS:
All outcomes underpowered (power < 50%, far below 80% standard).
Low power directly increases Type M/S errors.

EXAGGERATION HIERARCHY:
1. Stroop (Type M = 3.79×): EXTREME risk, do not trust estimates.
2. Tau (Type M = 4.90×): EXTREME risk, outlier-sensitive, unreliable.
3. PRP (Type M = 1.74×): MODERATE risk, interpret with caution.
4. WCST PE (Type M = 1.66×): MODERATE risk, most trustworthy of the four.

SIGN ERROR:
- WCST PE and PRP: 0% (sign stable when significant).
- Tau: 9.6% (HIGH risk of sign reversal).
- Stroop: 6.9% (moderate risk).

IMPLICATIONS FOR PUBLICATION:
1. Report Type M errors alongside p-values and effect sizes.
2. Acknowledge that observed effects likely overestimate true effects by
   66-490% depending on outcome.
3. For WCST PE (primary outcome): Report β=2.52 with caveat that true effect
   may be ~β=1.5-2.0 after correcting for winner's curse.
4. For tau and Stroop: Report as null/exploratory, not confirmatory.
5. URGENT: Plan replication with N ≥ 150 to achieve 80% power for β=1.76.

      estimates (Type M = 1.66-4.90×). Power analysis is NOT optional.

   E. Type M/S Errors Should Be Routine:
      Psychology has focused on Type I error (false positives) but ignored
