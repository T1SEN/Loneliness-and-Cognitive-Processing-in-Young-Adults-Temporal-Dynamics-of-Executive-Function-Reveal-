======================================================================
FRAMEWORK 2: NORMATIVE MODELING
Interpretation Guide
======================================================================

RESEARCH QUESTION:
----------------------------------------------------------------------
Does loneliness predict EF impairment BEYOND what is expected
given age, mood (DASS), and gender?

APPROACH:
----------------------------------------------------------------------
1. Built normative models: EF ~ f(Age, DASS, Gender)
2. Computed personalized deviations (Observed - Expected)
3. Tested: Deviation ~ UCLA * Gender

KEY RESULTS:
----------------------------------------------------------------------

WCST Perseverative Error Rate (N=234):
  Normative model R²: 0.998
  UCLA → Deviation: β = -0.004, p = 0.957
  UCLA × Gender: β = -0.067, p = 0.622
  Bootstrap 95% CI (UCLA): [-0.009, 0.000]
  Bootstrap 95% CI (Interaction): [-0.619, 0.504]

PRP Bottleneck Effect (ms) (N=234):
  Normative model R²: 0.992
  UCLA → Deviation: β = 0.108, p = 0.187
  UCLA × Gender: β = -0.110, p = 0.411
  Bootstrap 95% CI (UCLA): [-0.000, 0.254]
  Bootstrap 95% CI (Interaction): [-0.265, 0.022]

Stroop Interference (ms) (N=240):
  Normative model R²: 0.968
  UCLA → Deviation: β = 0.019, p = 0.817
  UCLA × Gender: β = -0.267, p = 0.044
  Bootstrap 95% CI (UCLA): [-0.001, 0.069]
  Bootstrap 95% CI (Interaction): [-0.666, 0.036]


INTERPRETATION:
----------------------------------------------------------------------
- Positive UCLA β: Loneliness predicts WORSE-than-expected EF
- Positive Interaction β: Effect stronger in males
- Negative UCLA β: Loneliness predicts BETTER-than-expected EF (unexpected)
- β near 0: No additional UCLA effect beyond demographics/mood

CLINICAL IMPLICATIONS:
----------------------------------------------------------------------
If UCLA β > 0 in males:
  → Lonely men show 'excessive' EF impairment not explained by mood
  → Early risk marker for cognitive vulnerability
  → Personalized normative approach identifies at-risk individuals

COMPARISON TO FRAMEWORK 1:
----------------------------------------------------------------------
- Framework 1 (Mixtures): Found latent subgroups with different slopes
- Framework 2 (Normative): Quantifies how much each person deviates
- Both converge if: High-deviation individuals = Vulnerable mixture class
