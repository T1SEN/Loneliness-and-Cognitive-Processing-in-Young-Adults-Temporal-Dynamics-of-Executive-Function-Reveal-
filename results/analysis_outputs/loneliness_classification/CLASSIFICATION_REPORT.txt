================================================================================
LONELINESS CLASSIFICATION REPORT
================================================================================

RESEARCH QUESTION
--------------------------------------------------------------------------------
Can executive function patterns predict high vs low loneliness?
If so, is this predictive signature gender-specific?

METHOD
--------------------------------------------------------------------------------
Target: High loneliness (UCLA ≥49.0) vs Low (UCLA ≤32.0)
N: 26 participants (extremes only)
Features: 11 EF metrics
Models: Logistic Regression + Random Forest
Validation: 5-fold stratified cross-validation

KEY FINDINGS
--------------------------------------------------------------------------------

Logistic Regression - Overall (N=26):
  AUC: 0.658 ± 0.180
  Accuracy: 0.647
  F1: 0.697

Logistic Regression - Female (N=19):
  AUC: 0.675 ± 0.089
  Accuracy: 0.627
  F1: 0.735

Random Forest - Overall (N=26):
  AUC: 0.508 ± 0.180
  Accuracy: 0.493
  F1: 0.609

Random Forest - Female (N=19):
  AUC: 0.862 ± 0.130
  Accuracy: 0.738
  F1: 0.841

INTERPRETATION GUIDE
--------------------------------------------------------------------------------
AUC = 0.50: Random chance (no signal)
AUC = 0.60-0.65: Weak predictive signal
AUC = 0.65-0.75: Moderate signal (meaningful individual differences)
AUC > 0.75: Strong signal (potential screening utility)

CONCLUSION
--------------------------------------------------------------------------------
✗ Overall AUC = 0.508 → WEAK/NO predictive signal.
  Current EF metrics do not reliably predict loneliness status.

LIMITATIONS
--------------------------------------------------------------------------------
- Small sample size limits generalizability
- Extreme group design (top/bottom 25%) may inflate AUC
- Cross-validation on same dataset (not external validation)
- Predictive model ≠ causal mechanism

NEXT STEPS
--------------------------------------------------------------------------------
- If AUC > 0.65: Cite in Discussion as evidence for measurable EF signature
- Examine feature importance to identify which EF metrics drive classification
- If gender-specific: Connect to interaction findings from regression models
- Future: External validation in independent sample
