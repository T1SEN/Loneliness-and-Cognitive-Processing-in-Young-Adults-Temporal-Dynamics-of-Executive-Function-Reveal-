  This supports a general 'meta-control' disruption rather than task-specific deficits.

THEORETICAL IMPLICATIONS
--------------------------------------------------------------------------------
Multiple EF tasks show coordinated vulnerability to loneliness (male-specific).
This suggests a DOMAIN-GENERAL meta-control mechanism, not isolated deficits.

Possible mechanisms:
  - Reduced proactive/sustained attention affecting all controlled tasks
  - Working memory disruption cascading across EF domains
  - Motivational/energetic depletion impacting effortful control globally
