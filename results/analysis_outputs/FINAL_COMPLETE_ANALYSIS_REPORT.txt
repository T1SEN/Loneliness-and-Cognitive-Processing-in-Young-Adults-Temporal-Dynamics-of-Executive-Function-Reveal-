**핵심 메시지**:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   • 외로움 + 낮은 각성 (Low DASS) = 취약성 (16%)
     → 주의 lapses 심각 (PRP τ 299ms vs 124ms)

   • 외로움 + 높은 각성 (High DASS) = 회복탄력성 (13%)
     → 보호적 hypervigilance (stress 18 vs 7)

   • RT 변동성(σ)은 취약성 **지표** (매개변인 아님!)
     → WCST-Stroop 공통 메커니즘 (r=0.544)
     → UCLA와 독립적, 직접 개입 타겟

   • UCLA > 44점 = 스크리닝 임계값 (민감도 62.5%)

   → 개인화된 평가 (UCLA + DASS + σ/PRP τ)와
      프로파일별 표적 개입이 가능합니다."

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   - 취약군 → 주의 훈련 + **RT 일관성 훈련** (σ 직접 타겟)
   - 회복탄력군 → 추적 관찰
4. **치료 차별화**: 항불안제 신중 사용 (보호적 불안 보존)

**학술적 가치**:
1. **Person-centered approach**: 집단→개인 수준 분석
2. **역설적 발견**: 불안/스트레스의 보호 효과
