EXECUTIVE SUMMARY
================================================================================

UCLA loneliness shows ROBUST, GENDER-SPECIFIC effects on dual-task coordination
that survive rigorous control for depression, anxiety, and stress.

MALE PATTERN: Attentional Lapse Amplification
  - Higher UCLA → More frequent attentional lapses (τ parameter)
  - Manifest as: Increased RT variability, greater bottleneck effect
  - Mechanism: Sustained attention failures under dual-task load

FEMALE PATTERN: Error Cascade Propagation
  - Higher UCLA → More error cascades (T1 error → T2 error)
  - Manifest as: Increased cascade rate and inflation
  - Mechanism: Impaired error monitoring and recovery

THEORETICAL SIGNIFICANCE:
These patterns parallel WCST findings, suggesting CROSS-TASK gender-specific
vulnerability pathways that are INDEPENDENT of mood/anxiety.

A. UCLA × GENDER INTERACTIONS (DASS Controlled, p < 0.05)
----------------------------------------------------------

1. Mean T2 RT: β = 114.49, p = 0.013**
   - Males: β = 85.43, p = 0.084
   - Females: β = 7.77, p = 0.867

2. PRP Bottleneck: β = 69.97, p = 0.034*
   - Males: β = 41.24, p = 0.216
   - Females: β = -7.54, p = 0.820
   → UCLA predicts greater dual-task interference in males

3. SOA Slope: β = -34.98, p = 0.034*
   - Males: β = -20.62, p = 0.216
   - Females: β = 3.77, p = 0.820
   → Males show steeper RT reduction with longer SOA

4. T2 RT Variability (IQR): β = 88.28, p = 0.016*
   - Males: β = 76.98, p = 0.032*
   - Females: β = 7.82, p = 0.827
   → UCLA predicts HIGHER RT inconsistency in males

B. FEMALE-SPECIFIC EFFECTS (DASS Controlled, p < 0.05)
------------------------------------------------------

1. Error Cascade Rate: β = 0.153, p = 0.030*
   - Females only (males: β = -0.036, p = 0.674)
   → UCLA predicts T1 error → T2 error propagation

2. Cascade Inflation: β = 0.146, p = 0.037*
   - Females only (males: β = -0.032, p = 0.712)
   → UCLA predicts excess T2 errors beyond base rate

C. MECHANISTIC INSIGHTS: EX-GAUSSIAN DECOMPOSITION
--------------------------------------------------

Overall RT = μ (Gaussian mean) + σ (Gaussian SD) + τ (Exponential tail)
  - μ: Routine processing speed (central tendency)
  - σ: Trial-to-trial variability (consistency)
  - τ: Attentional lapses (slow outlier trials)

Key Finding: MALE VARIABILITY DRIVEN BY τ (ATTENTIONAL LAPSES)

1. τ (Exponential tail): β = 88.84, p = 0.016*
   - Males: β = 63.88, p = 0.057
   - Females: β = -9.66, p = 0.795
   → Males show MORE attentional lapses with higher UCLA

2. τ at short SOA: β = 81.80, p = 0.049*
   - Males: β = 31.45, p = 0.489
   - Females: β = -75.93, p = 0.077
   → At high load (short SOA), males show more lapses

3. Δσ bottleneck: β = 47.49, p = 0.093 (marginal)
   - Suggests both σ and τ contribute to variability
   → But τ effect is stronger and more significant

INTERPRETATION:
- Male RT variability is NOT general inconsistency (σ)
- Instead: Specific ATTENTIONAL LAPSE events (τ)
- These lapses are AMPLIFIED under dual-task load (short SOA)

IV. CLINICAL & PRACTICAL IMPLICATIONS
================================================================================

A. ASSESSMENT RECOMMENDATIONS
------------------------------

For MALES with high loneliness:
  Standard Assessment:
    ❌ Mean RT (may appear normal)
    ❌ Overall accuracy (may appear normal)

  Sensitive Markers:
    ✅ RT variability (IQR, CV, or Ex-Gaussian τ)
    ✅ Bottleneck effect magnitude
    ✅ Performance under high cognitive load

  Clinical Red Flags:
    - High RT variability despite normal mean RT
    - Inconsistent performance across trials
    - Large bottleneck effect (> 600 ms)
    - Frequent attentional lapses (high τ)

For FEMALES with high loneliness:
  Standard Assessment:
    ❌ Mean error rate (may appear normal due to low base rate)
    ❌ Single-trial accuracy

  Sensitive Markers:
    ✅ Error cascade rate (P(T2 error | T1 error))
    ✅ Sequential error dependencies
    ✅ Post-error performance degradation

  Clinical Red Flags:
    - Normal overall accuracy but HIGH cascade rate
    - Errors cluster in sequences
    - Lack of post-error RT adjustment

B. INTERVENTION STRATEGIES
---------------------------

For MALES: Attention Stabilization Training
  Target: Reduce attentional lapses (τ) under dual-task load

  Interventions:
    1. Sustained Attention Training
       - Progressive dual-task difficulty
       - Focus on maintaining CONSISTENT performance
       - Reward stability, not just speed

    2. Resource Allocation Training
       - Teach explicit dual-task strategies
       - Practice prioritization under load
       - Reduce reactive, increase proactive control

    3. Metacognitive Awareness
       - Self-monitoring of performance variability
       - Identify lapse triggers (fatigue, distraction)
       - Develop lapse-prevention strategies

  Outcome Metrics:
    - Reduced RT variability (IQR, τ)
    - Smaller bottleneck effect
    - More consistent dual-task performance

For FEMALES: Error Monitoring Enhancement
  Target: Reduce error cascades (T1 error → T2 error)

  Interventions:
    1. Error Awareness Training
       - Immediate feedback on errors
       - Teach error detection before external feedback
       - Enhance internal monitoring signals

    2. Post-Error Recovery Training
       - Explicit post-error strategies (pause, refocus)
       - Practice recovering from errors
       - Reduce error-induced anxiety/rumination

    3. Sequential Performance Monitoring
       - Track trial-by-trial performance
       - Identify error clusters
       - Develop cluster-breaking strategies

  Outcome Metrics:
    - Reduced error cascade rate
    - Increased post-error slowing (adaptive)
    - Reduced cascade inflation

V. LIMITATIONS & FUTURE DIRECTIONS
================================================================================

A. CURRENT LIMITATIONS
-----------------------

1. Sample Size:
   - Error cascade N=33 (limited power)
   - Male subsample N=30 (adequate but not large)
   - Need N ≥ 100 per gender for robust interactions

2. Cross-Sectional Design:
   - Cannot infer causality (UCLA → PRP or PRP → UCLA)
   - Cannot rule out third-variable confounds
   - Need longitudinal design

3. Multiple Comparisons:
   - 11 DVs tested, some p-values marginal (0.03-0.05)
   - Would not all survive strict Bonferroni correction
   - Recommend replication before strong claims

4. Mechanism Unclear:
   - Ex-Gaussian provides descriptive mechanistic insight
   - But neural basis unknown (need EEG/fMRI)
   - Causal pathways uncertain (need intervention studies)

5. Ecological Validity:
   - Laboratory PRP task ≠ real-world dual-tasking
   - Need to validate against functional outcomes
   - Does τ or cascade rate predict daily multitasking?

B. RECOMMENDED FOLLOW-UP ANALYSES
----------------------------------

1. Trial-Level Mixed-Effects Models
   Purpose: More power than participant-level aggregation
   DV: Trial-level T2 RT or accuracy
   Fixed effects: UCLA × Gender × SOA × Trial number
   Random effects: Participant intercepts and slopes
   Benefits: Can test time-on-task, fatigue, learning

2. Cross-Task Correlation Analysis
   Test: Does PRP bottleneck correlate with WCST PE (within males)?
   Test: Does PRP cascade rate correlate with WCST error persistence (within females)?
   Purpose: Validate "common executive fragility" hypothesis

3. Mediation Analysis
   Question: Does PRP variability (τ) mediate UCLA → functional outcomes?
   Path: UCLA → PRP τ → Daily multitasking failures (males)
   Path: UCLA → Cascade rate → Daily error proneness (females)

4. Moderator Analysis
   Test: Does DASS moderate the UCLA → PRP relationship?
   Question: Are effects stronger at high vs. low DASS?
   Alternative: Do effects exist ONLY at high DASS (not truly independent)?

5. Sequential Error Analysis (Females)
   Test: Transition matrices (P(error_trial N+1 | error_trial N))
   Test: Run-length analysis (how long do error sequences last?)
   Purpose: Characterize cascade dynamics in more detail

C. RECOMMENDED REPLICATION STUDY
---------------------------------

Design:
  - N ≥ 100 per gender (total N ≥ 200)
  - Pre-registered hypotheses (prevent p-hacking)
  - Primary DVs: Bottleneck, RT variability, error cascade rate
  - Add neural measures: EEG (ERN for females, CNV for males)
  - Add functional outcomes: Real-world dual-tasking questionnaire

Pre-Registered Hypotheses:
  H1: Males show UCLA × Gender → PRP bottleneck (p < 0.05)
  H2: Males show UCLA × Gender → T2 RT variability (IQR) (p < 0.05)
  H3: Males show UCLA × Gender → τ (attentional lapses) (p < 0.05)
  H4: Females show UCLA → Error cascade rate (p < 0.05)
  H5: PRP patterns correlate with WCST patterns within gender

Power Analysis:
  - For interaction R² = 0.10, need N ≥ 160 (α = 0.05, power = 0.80)
  - For gender-stratified R² = 0.15, need N ≥ 55 per group
