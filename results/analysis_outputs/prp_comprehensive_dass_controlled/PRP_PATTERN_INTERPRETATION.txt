II. KEY FINDINGS: GENDER-MODERATED UCLA EFFECTS
================================================================================

A. PRIMARY INTERACTIONS (UCLA × Gender, DASS controlled)
------------------------------------------------------

1. PRP Bottleneck Effect (T2 RT: short SOA - long SOA)
   ✅ Interaction: β = 69.97, p = 0.034
   - Males show POSITIVE association (β = 41.24, p = 0.216, trend)
   - Females show near-zero association (β = -7.54, p = 0.820)

   INTERPRETATION: Higher UCLA loneliness → Greater dual-task interference in males

2. Mean T2 RT (Overall dual-task speed)
   ✅ Interaction: β = 114.49, p = 0.013**
   - Males: β = 85.43, p = 0.084 (marginal)
   - Females: β = 7.77, p = 0.867

   INTERPRETATION: UCLA predicts slower dual-task processing specifically in males

3. SOA Slope (Linear trend in T2 RT across SOA bins)
   ✅ Interaction: β = -34.98, p = 0.034
   - Males: β = -20.62, p = 0.216 (steeper negative slope)
   - Females: β = 3.77, p = 0.820

   INTERPRETATION: Males show greater RT benefit from longer SOA preparation time

4. T2 RT Variability (IQR)
   ✅ Interaction: β = 88.28, p = 0.016*
   ✅ Males: β = 76.98, p = 0.032*
   - Females: β = 7.82, p = 0.827

   INTERPRETATION: UCLA predicts HIGHER RT variability (inconsistency) in males

================================================================================

B. FEMALE-SPECIFIC EFFECTS (DASS controlled)
-------------------------------------------

1. Error Cascade Rate (T1 error → T2 error probability)
   ✅ Main effect: β = 0.107, p = 0.017*
   ✅ Females only: β = 0.153, p = 0.030*
   - Males: β = -0.036, p = 0.674 (no effect)

   INTERPRETATION: UCLA predicts error propagation specifically in females

2. Cascade Inflation (Excess T2 errors beyond base rate)
   ✅ Main effect: β = 0.094, p = 0.037*
   ✅ Females only: β = 0.146, p = 0.037*
   - Males: β = -0.032, p = 0.712 (no effect)

   INTERPRETATION: Females show cascading failures under dual-task load

III. THEORETICAL PATTERN: PARALLEL TO WCST FINDINGS
================================================================================

WCST PATTERN (Your Previous Finding):
--------------------------------------
- Males: UCLA → Higher perseverative errors (set-shifting rigidity)
- Females: UCLA → Error cascades and monitoring failures

PRP PATTERN (Current Analysis):
-------------------------------
- Males: UCLA → Higher RT variability + bottleneck effect (dual-task instability)
- Females: UCLA → Error cascades (T1 error → T2 error propagation)

UNIFIED INTERPRETATION:
-----------------------

1. MALE VULNERABILITY: Performance Variability
   - WCST: Perseverative errors (cannot shift flexibly)
   - PRP: RT variability + bottleneck effect (inconsistent dual-task coordination)
   - Mechanism: INSTABILITY in executive control under cognitive load

   Theory: Lonely males show REACTIVE control deficits
   → Cannot maintain stable performance under interference/load
   → Errors manifest as inconsistency (high variability)

2. FEMALE VULNERABILITY: Error Propagation
   - WCST: Post-error adaptation failures
   - PRP: Error cascades (T1 error → T2 error)
   - Mechanism: MONITORING failures lead to cascading errors

   Theory: Lonely females show MONITORING/RECOVERY deficits
   → Can maintain average performance
   → But errors cascade when monitoring lapses
   → Suggests impaired error detection or correction

IV. STATISTICAL ROBUSTNESS
================================================================================

A. Effect Sizes (R² adjusted):
   - PRP Bottleneck × Gender: R²adj = 0.269
   - Mean T2 RT × Gender: R²adj = 0.336
   - T2 IQR × Gender: R²adj = 0.278
   - Male T2 IQR: R²adj = 0.520 (large effect!)

B. Sample Sizes:
   - Full sample interactions: N = 75 (adequate power)
   - Error cascade analyses: N = 33 (limited, requires replication)
   - Gender-stratified: Females N=45, Males N=30 (adequate)

C. Multiple Comparison Considerations:
   - 11 DVs tested → Bonferroni correction: α = 0.05/11 = 0.0045
   - Interactions p < 0.034 would NOT survive strict Bonferroni
   - However, interactions show CONSISTENT pattern (males: variability, females: errors)
   - Recommend treating as exploratory findings requiring replication

V. MECHANISTIC HYPOTHESES
================================================================================

A. MALE PATHWAY: Dual-Task Coordination Instability

   Hypothesis 1: Central Bottleneck Amplification
   - Loneliness → Reduced cognitive resources
   - Under dual-task load → Bottleneck becomes MORE variable
   - Manifests as: High RT variability, greater bottleneck effect

   Hypothesis 2: Proactive Control Failure
   - Loneliness impairs proactive preparation
   - Cannot use long SOA to prepare effectively
   - Leads to inconsistent dual-task performance

   Next Steps:
   - Ex-Gaussian decomposition: Is variability in μ (mean) or τ (lapses)?
   - Trial-level analysis: Does variability increase over time (fatigue)?
   - Cross-task correlation: Does PRP variability correlate with WCST PE?

B. FEMALE PATHWAY: Error Monitoring Cascade

   Hypothesis 1: Impaired Error Detection
   - T1 error goes undetected
   - No compensatory adjustment
   - T2 error follows

   Hypothesis 2: Resource Depletion After Error
   - T1 error depletes cognitive resources
   - Insufficient resources for T2
   - Cascade occurs

   Next Steps:
   - Post-error RT analysis: Do females show reduced post-error slowing?
   - ERP markers: Is error-related negativity (ERN) reduced?
   - Cross-task: Do PRP error cascades correlate with WCST error persistence?

VI. CLINICAL/APPLIED IMPLICATIONS
================================================================================

A. Assessment Implications:

   For Males:
   - Standard mean RT may UNDERESTIMATE dual-task impairment
   - RT variability (IQR, CV) is MORE sensitive marker
   - Recommend: Report RT variability alongside mean RT

   For Females:
   - Error rate may appear normal (low base rate)
   - Error CASCADE rate is more sensitive marker
   - Recommend: Analyze trial-by-trial error dependencies

B. Intervention Implications:

   For Males:
   - Target: Stabilizing dual-task performance
   - Strategy: Training proactive control, resource allocation
   - Metric: Reduce RT variability, not just mean RT

   For Females:
   - Target: Error monitoring and recovery
   - Strategy: Metacognitive training, error awareness
   - Metric: Reduce error cascade rate

VII. LIMITATIONS & FUTURE DIRECTIONS
================================================================================

A. Current Limitations:

   1. Sample Size:
      - Error cascade N=33 (requires ≥3 T1 errors per participant)
      - Limits power for cascade interaction tests
      - Need larger sample or easier task to increase error rate

   2. Cross-Sectional Design:
      - Cannot determine causality (UCLA → PRP or PRP → UCLA)
      - Cannot rule out third-variable confounds

   3. Multiple Comparisons:
      - 11 DVs tested, some p-values marginal (0.03-0.05)
      - Requires replication in independent sample

   4. Mechanism Unclear:
      - Need Ex-Gaussian, trial-level mixed models
      - Need neural correlates (EEG, fMRI)

B. Recommended Follow-Up Analyses:

   1. Ex-Gaussian RT Decomposition (WITH DASS controls):
      - Decompose RT into μ (mean), σ (variability), τ (lapses)
      - Test: Does male variability reflect σ or τ?

   2. Trial-Level Mixed-Effects Models:
      - Nested structure: Trials within participants
      - Test time-on-task effects: Does variability increase?
      - Test sequential dependencies: Error cascades

   3. Cross-Task Integration:
      - Correlation: PRP bottleneck × WCST PE (within males)
      - Correlation: PRP error cascade × WCST error persistence (within females)
      - Test: Common "executive fragility" factor?

   4. Mediation Analysis:
      - Does PRP variability mediate UCLA → functional outcomes?
      - Does error cascade rate mediate UCLA → daily functioning?

C. Recommended Replication Study:

   Design:
   - N ≥ 100 per gender (adequate power)
   - Add easier PRP task to increase error rate
   - Add neural markers (EEG: ERN, P3b)
   - Add daily functioning measures (real-world dual-tasking)

   Pre-registered Hypotheses:
   1. Males: UCLA × Gender → PRP bottleneck effect (p < 0.05)
   2. Males: UCLA × Gender → T2 RT variability (p < 0.05)
   3. Females: UCLA × Gender → Error cascade rate (p < 0.05)

- Supports: Multiple dissociable executive control mechanisms

- Standard metrics (mean RT, mean accuracy) may miss gender-specific effects
- Males: Need variability-based assessment
- Females: Need error cascade-based assessment
- Interventions should be gender-tailored
