**생성 날짜**: 2025-11-15

## 📊 **핵심 발견 요약**

### 🎯 **주요 효과 (Main Finding)**

**성별 조절효과 (Gender Moderation)** - 견고하고 재현 가능
```
UCLA × Gender → WCST 보속오류
- 상호작용: β = 2.59, p = 0.004
- Permutation test: p = 0.004
- Bootstrap 95% CI: [0.80, 4.50] (99.6% positive)
```

**성별별 단순 기울기 (Simple Slopes)**:
- **남성**: β = 2.29, p = 0.068 (marginal)
  - 해석: 외로움 ↑ → 보속오류 ↑ (인지 경직성)
- **여성**: β = -0.30, p = 0.724 (ns)
  - 해석: 외로움과 무관

---

### ❌ **주효과 부재 (Null Main Effects)** - 이론적으로 중요

**실행기능 과제별 UCLA 상관**:
- Stroop 간섭: r = 0.029, p = 0.81
- WCST 보속오류: r = -0.100, p = 0.41
- PRP 병목: r = 0.054, p = 0.65

**왜 주효과가 없는가?**
1. **이질성 (Heterogeneity)**: 성별 조절로 인한 효과 상쇄
2. **낮은 신뢰도**: 과제 신뢰도 0.54-0.60 → 40-48% 감쇠
3. **보상 과정 (Compensation)**: 여성의 보상 노력 (반대 방향 효과)
4. **맥락 의존성**: 특정 인지 요구에서만 나타남

## 🔬 **신규 심층 분석 결과** (2025-11-15 완료)

### **분석 1: 매개 분석 (Mediation Analysis)**

**연구 질문**: 성별 조절효과가 **왜** 발생하는가? DASS (우울/불안/스트레스)가 매개하는가?

**주요 발견**:
- ❌ **유의미한 매개 없음** (모든 95% CI가 0 포함)
- ✅ **반대 방향 패턴 발견**:

| 성별 | 경로 | 간접효과 | 95% CI | 해석 |
|------|------|----------|--------|------|
| 남성 | UCLA → 우울 → PE | -1.04 | [-3.81, 0.93] | 억제 효과 |
| 남성 | UCLA → 불안 → PE | -0.89 | [-2.47, 0.19] | 억제 효과 |
| 남성 | UCLA → 스트레스 → PE | -1.60 | [-4.07, 0.43] | 억제 효과 |
| 여성 | UCLA → 우울 → PE | -0.51 | [-1.88, 0.51] | 미미 |
| 여성 | UCLA → 불안 → PE | +0.23 | [-0.60, 1.40] | 역방향 |
| 여성 | UCLA → 스트레스 → PE | -0.10 | [-1.31, 0.88] | 미미 |

**이론적 함의**:
- UCLA와 DASS가 **독립적 경로**로 WCST에 영향 (매개 아님)
- **억제 효과 (Suppression)**: DASS 통제 시 UCLA 효과 증가
- UCLA는 "사회적 고립", DASS는 "정서적 고통" - 별개 구성개념

### **분석 2: 3방향 상호작용 (Three-Way Interactions)**

**연구 질문**: 성별 조절효과가 나이/교육/DASS 수준에 따라 달라지는가?

**결과**:

| 조절변수 | ΔR² | F | p | 유의성 |
|----------|-----|---|---|--------|
| UCLA × Gender × **Age** | 0.027 | 1.93 | 0.170 | Marginal |
| UCLA × Gender × Education | 0.000 | 0.00 | 1.000 | ns |
| UCLA × Gender × DASS | 0.000 | 0.00 | 1.000 | ns |

**나이 조절 패턴** (marginal, p=0.17):
- 젊은 남성: β = +1.230 (외로움 → PE ↑)
- 젊은 여성: β = -1.318 (외로움 → PE ↓)
- **문제**: 나이 많은 참가자 N=6 → 검증력 부족

**한계**:
- 나이 범위 좁음 (대부분 20대)
- 교육 변량 부족 (거의 모두 대학생)
- DASS 교차효과 없음

### **분석 3: 오류 유형 분석 (PE vs. NPE)** ⭐ **핵심 검증**

**연구 질문**: 외로움이 **인지 경직성** (PE) 특이적인가, **일반 손상** (NPE도 증가)인가?

**결과**:

| 오류 유형 | UCLA × Gender 상호작용 β | 남성 r | 여성 r |
|-----------|-------------------------|--------|--------|
| **PE (보속오류)** | **2.157** | +0.236 | -0.232 |
| **NPE (비보속오류)** | **1.105** | +0.034 | -0.118 |

**해석**:
- ✅ **PE 상호작용이 NPE보다 2배 강함** → 경직성 특이적
- ✅ 남성: UCLA → PE (r=0.236) but NOT NPE (r=0.034)
- ✅ **경직성 가설 검증**: 외로움은 유연성 손상, 일반 능력은 정상

**추가 발견**:
- **PE 연속 길이** (평균 PE run length): 여성에서 UCLA와 유의 상관 (r=-0.376, p=0.01)
  - 외로운 여성은 오히려 보속 깊이가 **짧음** (빠른 전환?)

### **분석 4: UCLA 하위요인 분석 (Facet Analysis)** ⭐ **구성개념 정교화**

**연구 질문**: **사회적** 외로움 vs. **정서적** 외로움 중 어느 것이 성별 조절효과를 만드는가?

**요인 분석 결과**:
- Factor 1: Social Loneliness (사회적 네트워크 부족)
- Factor 2: Emotional Loneliness (친밀한 관계 부족)

**성별별 상관**:

| 하위요인 | 남성 | 여성 | 상호작용 β |
|----------|------|------|------------|
| **정서적 외로움** | r=0.358, **p=0.062** | r=-0.183 | **2.389** |
| 사회적 외로움 | r=0.270, p=0.165 | r=-0.231 | 2.303 |
| 총점 | r=0.236, p=0.227 | r=-0.232 | 2.157 |

**핵심 발견**:
- ✅ **정서적 외로움**이 약간 더 강력한 성별 조절효과
- ✅ 남성의 정서적 외로움 효과가 **marginal significance** (p=0.062)
- ❌ 초기 가설 (사회적 고립 → 피드백 학습 결핍) 부분 지지
- ✅ **수정된 해석**: 친밀한 정서적 연결 부족이 남성에게 특히 문제
  - 남성성 규범 위배 (남성은 감정적으로 독립적이어야 함)
  - 하지만 실제로는 정서적 지지 필요 → 부족 시 인지 경직성

**예측력 비교**:
- 총점 R² = 0.0076
- 하위요인 R² = 0.0063 (거의 동일)
- 하위요인이 추가 설명력 제공하지 않음 (구성개념 이해에는 유용)

## 🧩 **통합 이론 모델**

### **맥락 의존적 인지 취약성 (Context-Dependent Cognitive Vulnerability)**

```
┌─────────────────────────────────────────────────────────────────┐
│                  외로움의 성별 특이적 경로                        │
└─────────────────────────────────────────────────────────────────┘

남성 경로 (Vulnerable):
외로움 (특히 정서적)
  → 남성성 규범 위배 (감정적 독립 기대)
  → 정서적 고통 ↑ (DASS)
  → 인지 경직성 (반추/정신적 보속)
  → WCST 보속오류 ↑ (행동적 보속)

여성 경로 (Resilient):
외로움
  → Tend-and-Befriend 대처 (사회적 지지 추구)
  → 보상 노력 (metacognitive monitoring)
  → WCST 수행 유지 or 향상
```

### **왜 WCST만 영향받는가?**

| 과제 특성 | Stroop | PRP | **WCST** |
|-----------|--------|-----|----------|
| 피드백 학습 | ✗ | ✗ | **✓** |
| 유연한 전환 | ✗ | ✗ | **✓** |
| 작업기억 부하 | 낮음 | 중간 | **높음** |
| 메타인지 모니터링 | ✗ | ✗ | **✓** |

**WCST 특이성 설명**:
- 시행별 피드백 통합 필요 (사회적 학습과 유사)
- 숨겨진 규칙 발견 → 가설 검증 → 유연한 수정
- 외로운 남성: 피드백 처리 손상 + 경직된 사고 → 보속

**권장사항**:

## 💡 **이론적 기여 요약**

### **3가지 핵심 통찰**:

1. **성별 조절의 메커니즘**
   - 남성성 규범 위배 + 정서적 지지 부족 → 인지 경직성
   - 여성의 보상 전략 (tend-and-befriend)

2. **과제 특이성의 이유**
   - 피드백 학습 + 유연성 요구 = WCST 특이성
   - 사회적 학습 결핍이 피드백 기반 과제에 전이

3. **구성개념 정교화**
   - 외로움 ≠ 우울/불안 (독립 경로)
   - 정서적 외로움 > 사회적 외로움 (남성)
   - PE (경직성) ≠ NPE (일반 손상)
