## 🚩 중요한 제약사항 및 주의사항

### 🚩 Red Flag 1: 다중비교 미보정

**문제**:
- HMM에서 5개 DV 테스트 → 1개만 유의 (20% 적중률)
- 기회 수준(α=0.05)에서 예상되는 적중률과 유사
- Bonferroni: p=0.017 > α=0.01 → **통과 못함**

**권장 조치**:
```

"이 발견은 탐색적 분석에서 나온 것이며 family-wise error rate

α=0.01, observed p=0.017). 따라서 이는 가설 생성적 발견으로
간주되어야 하며 독립 표본에서의 사전 등록된 복제 연구가 필요하다."
```

