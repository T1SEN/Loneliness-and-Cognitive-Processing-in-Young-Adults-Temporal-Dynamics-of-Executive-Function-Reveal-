================================================================================
RESIDUAL UCLA ANALYSIS REPORT
================================================================================

PURPOSE
--------------------------------------------------------------------------------
Test whether UCLA loneliness effects persist after removing variance
explained by DASS-21 (depression/anxiety/stress).

This addresses the question: Are we measuring 'social-cognitive loneliness'
or 'affective/dysphoric loneliness'?

METHOD
--------------------------------------------------------------------------------
1. Regressed UCLA on DASS subscales (R² = 0.475)
   → DASS explains 47.5% of UCLA variance

2. Extracted residuals → UCLA_resid (orthogonal to DASS by construction)

3. Re-ran key models:
   Original:  outcome ~ UCLA × Gender + DASS + Age
   Residual:  outcome ~ UCLA_resid × Gender + DASS + Age

KEY FINDINGS
--------------------------------------------------------------------------------

WCST Perseverative Error Rate
  N = 150
  Original UCLA × Gender:  β=0.1923, p=0.2616
  Residual UCLA × Gender:  β=0.1069, p=0.5465
  → NOT SIGNIFICANT: No interaction in original model

PRP Bottleneck Effect
  N = 154
  Original UCLA × Gender:  β=0.2434, p=0.1377
  Residual UCLA × Gender:  β=0.0728, p=0.6708
  → NOT SIGNIFICANT: No interaction in original model

Stroop Interference
  N = 158
  Original UCLA × Gender:  β=0.0934, p=0.5801
  Residual UCLA × Gender:  β=0.1799, p=0.3052
  → NOT SIGNIFICANT: No interaction in original model


INTERPRETATION GUIDE
--------------------------------------------------------------------------------
SURVIVES: Effect independent of affective distress (pure social loneliness)
ELIMINATED: Effect was confounded with depression/anxiety/stress
EMERGED: Suppression effect - DASS was masking true social loneliness effect

THEORETICAL IMPLICATIONS
--------------------------------------------------------------------------------
No significant UCLA × Gender interactions found in either model.
Current sample may lack power to detect residual effects.


CONCLUSION
--------------------------------------------------------------------------------
Mixed pattern: Effects vary by outcome measure.
Some EF domains show pure social loneliness sensitivity,
others are confounded with general affective distress.
