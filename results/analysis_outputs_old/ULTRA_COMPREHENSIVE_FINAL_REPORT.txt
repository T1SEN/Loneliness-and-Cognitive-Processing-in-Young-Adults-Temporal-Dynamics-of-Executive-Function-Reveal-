**참가자**: N=82 (남성 31, 여성 51)

**결과**:

┌──────────┬─────────────┬──────────┐

**해석**:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

신규 분석 7: 시간 경과/인지 피로 분석
================================================================================

**연구 질문**: 외로운 남성이 과제 중 더 빨리 성능 저하를 보이는가?

**방법**:
- Trial-by-trial PE rate 변화 분석
- 개인별 fatigue slope 계산 (linear regression)
- Moving window analysis (10-trial windows)
- Fatigue slope × PRP τ 상관

**참가자**: N=82, 7,184 trials

**결과**:

**개인별 Fatigue Slopes**:
┌────────────────┬───────┬─────────────┐
│ Group          │ N     │ Slope       │
├────────────────┼───────┼─────────────┤
│ Lonely Males   │ 16    │ +0.000067   │
│ Non-lonely M   │ 15    │ +0.000086   │
│ Difference     │ -     │ -0.000019   │
│ **p-value**    │ -     │ **0.9564**  │
└────────────────┴───────┴─────────────┘

→ **NO DIFFERENCE**: 피로 기울기 차이 없음!

**Moving Window Results** (Males):
- Trials 1-10:   Lonely 0.0% vs Non-lonely 0.0%
- Trials 11-20:  Lonely 18.8% vs Non-lonely 14.0% (+4.7%)
- Trials 41-50:  Lonely 15.6% vs Non-lonely 10.0% (+5.6%)
- Trials 71-80:  Lonely 16.9% vs Non-lonely 8.6% (+8.3%)

→ 후반부 차이 약간 증가하나 통계적 유의성 없음

**Fatigue Slope × PRP τ**:
- r = -0.012, **p = 0.9545** (NS)

**해석**:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"**중요한 NULL FINDING**:

• PE 효과는 **시간 의존적이지 않음**
• 외로운 남성이 더 빨리 '소진'되지 않음
• PE rate가 trial 전반에 걸쳐 **안정적**
• PRP τ (주의 lapses)와 피로 무관

→ **메커니즘 재해석**:
   - PE는 **지속적 특성** (trait), 상태 변화 아님
   - '인지 피로' 가설 기각
   - 대신 **기저 주의 능력 결핍**으로 해석
   - 개입: 휴식/간격보다는 **지속 주의 훈련** 필요"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

신규 분석 8: 다변량 표현형 클러스터링

**방법**:
- K-means clustering (K=2~6 테스트)
- Features: PE_rate, WCST_SD, Stroop_σ, Stroop_τ, PRP_τ_long, PRP_σ_long
- Silhouette score로 최적 K 선택
- Cluster × UCLA/Gender 분포 분석

**참가자**: N=81 (남성 31, 여성 50)

**결과**:

**최적 K = 3** (Silhouette = 0.227)

**Cluster 0: "High PE (Vulnerable)"** - N=6 (33% 남성)
┌────────────────┬──────────┐
│ Metric         │ Value    │
├────────────────┼──────────┤

│ Stroop σ       │ 200.48   │
│ PRP τ (long)   │ 243.32ms │
│ UCLA           │ 33.0     │
│ DASS (all)     │ Low      │
└────────────────┴──────────┘
→ **최고 취약군**: 높은 PE, 낮은 정확도, 변동성 심함

**Cluster 1: "High τ (Attention Lapses)"** - N=29 (21% 남성)
┌────────────────┬──────────┐
│ Metric         │ Value    │
├────────────────┼──────────┤
│ PE rate        │ 11.08%   │

│ WCST SD        │ 2152.19  │
│ UCLA           │ 41.6     │
└────────────────┴──────────┘
→ **주의 lapses 중심**: 높은 τ, WCST 변동성

**Cluster 2: "High σ (Variable RT)"** - N=46 (50% 남성)
┌────────────────┬──────────┐
│ Metric         │ Value    │
├────────────────┼──────────┤
│ PE rate        │ 9.22%    │

**Cluster × UCLA Distribution (Males)**:
```
                 High UCLA    Low UCLA
Cluster 0           2            0
Cluster 1           5            1
Cluster 2           8           14
```

**Chi-square**: χ²=6.80, p=0.147 (NS, 표본 크기 제한)

**외로운 남성의 분포**:
- **53% (8/15)가 Cluster 2** (High σ phenotype)
- 33% (5/15)가 Cluster 1 (High τ phenotype)
- 13% (2/15)가 Cluster 0 (Vulnerable phenotype)

**해석**:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"**3개 구별되는 인지 표현형 발견**:

• **Vulnerable phenotype** (N=6, 7.4%):
  - 전반적 EF 손상
  - 높은 PE, 낮은 정확도
  - 다중 영역 개입 필요

• **Attention Lapses phenotype** (N=29, 35.8%):
  - PRP τ 특이적 손상
  - WCST RT 변동성 높음
  - Sustained attention 훈련 필요

• **Variable RT phenotype** (N=46, 56.8%):
  - PRP σ 특이적
  - 비교적 양호한 전반 성능
  - RT 일관성 훈련 필요

**외로운 남성의 과반 (53%)이 Variable RT phenotype**:
  - 이는 분석 4 발견과 일치 (WCST PE ↔ Stroop σ, r=0.544)
  - σ가 핵심 취약 지표 재확인
  - **개인 맞춤 개입** 가능:
    * Phenotype 1 → 종합 개입
    * Phenotype 2 → 지속 주의 훈련
    * Phenotype 3 → RT 일관성 훈련"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

신규 분석 9: 보속 모멘텀 분석

**연구 질문**: PE가 연속으로 뭉쳐있는가 (진짜 경직성) vs 무작위 분산 (일반 부주의)?

**방법**:
- PE run length 분석 (연속 PE의 길이)
- Lag-1 autocorrelation (PE[t] × PE[t+1])
- UCLA 상관: Run length vs PE rate 비교

**참가자**: N=82, 7,184 trials

**결과**:

**PE Run Metrics (Males)**:
┌────────────────┬───────┬──────────┬───────────┬──────────────┐
│ Group          │ N     │ Max Run  │ Mean Run  │ % in Runs≥2  │
├────────────────┼───────┼──────────┼───────────┼──────────────┤
│ Lonely         │ 15    │ 2.6      │ 1.26      │ 30.5%        │
│ Non-lonely     │ 16    │ 1.8      │ 1.16      │ 21.6%        │
├────────────────┼───────┼──────────┼───────────┼──────────────┤
│ t-test         │ -     │ t=1.610  │ t=1.215   │ t=1.111      │
│ **p-value**    │ -     │ **0.118**│ **0.234** │ **0.276**    │
└────────────────┴───────┴──────────┴───────────┴──────────────┘

→ **NO SIGNIFICANT DIFFERENCE**

**Autocorrelation (Males)**:
┌────────────────┬───────┬─────────────┬──────────┐
│ Group          │ N     │ r_lag1      │ p-value  │
├────────────────┼───────┼─────────────┼──────────┤
│ Lonely         │ 15    │ +0.069      │ -        │
│ Non-lonely     │ 16    │ +0.031      │ -        │
│ t-test         │ -     │ t=0.769     │ **0.448**│
└────────────────┴───────┴─────────────┴──────────┘

→ **NO DIFFERENCE**

**UCLA Correlations (Males, N=31)**:
┌─────────────────┬──────────┬──────────┐
│ Predictor       │ r        │ p        │
├─────────────────┼──────────┼──────────┤
│ **PE rate**     │ **+0.225**│ **0.224**│
│ Max run         │ +0.113   │ 0.545    │
│ Mean run        │ +0.034   │ 0.857    │
│ Autocorr        │ -0.020   │ 0.914    │
└─────────────────┴──────────┴──────────┘

→ **PE rate가 가장 강한 예측변인**

**해석**:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"**중요한 NULL FINDING #2**:

• PE가 연속으로 뭉쳐있지 않음 (run length 차이 NS)
• Lag-1 autocorrelation 차이 없음 (r=+0.07 vs +0.03)
• Run metrics가 UCLA 예측력 없음

→ **메커니즘 재해석**:
   - 외로운 남성의 PE는 **"진짜 보속 경직성"이 아님**
   - "Mental set에 갇힘" (stuck) 아님
   - 대신 **"일반적 부주의"** (randomly distributed errors)
   - PE들이 독립적으로 발생 (no momentum)

→ **임상적 함의**:
   - Set-shifting 특이적 훈련 불필요
   - 대신 **일반 주의력 훈련**이 적합
   - "과제 전환 능력" 문제 아님
   - "지속 주의 유지" 문제

→ **이론적 함의**:
   - WCST PE를 "perseverative rigidity"로 해석하는 것 재고
   - 외로움 맥락에서는 **attention deficit** 반영
   - Task-specific mechanism 아님"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
