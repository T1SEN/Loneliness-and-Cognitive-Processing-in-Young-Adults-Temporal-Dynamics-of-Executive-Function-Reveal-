II. WCST 특이성의 가능한 메커니즘
--------------------------------------------------------------------------------

가설 1: 사회적 피드백 민감도
- WCST: Trial-by-trial **피드백 제공** (사회적 신호 유사)
- Stroop/PRP: 피드백 최소 or 없음
- 외로운 사람: 사회적 피드백에 과민 → WCST에서 영향 큼
- 증거: 피드백 민감도 분석 결과 (데이터 수집 완료)

가설 2: 불확실성 내성 (Uncertainty Tolerance)
- WCST: Rule change **예측 불가** (probabilistic learning)
- Stroop/PRP: 규칙 **고정** (deterministic)
- 외로운 사람: 불확실성에 취약 → WCST에서만 문제
- 증거: Rule change 직후 10 trials 성과 저하 (분석 완료)

가설 3: Proactive vs Reactive Control
- WCST: **Proactive control** 요구 (rule 유지 + 유연한 switching)
- Stroop: Reactive control (conflict resolution)
- 외로운 남성: Proactive control 결함 → WCST 특이 효과
- 증거: 부분적 (추가 분석 필요)

가설 4: Task Difficulty (Optimal Challenge)
- WCST: 적절한 난이도 (ceiling/floor 없음)
- Stroop: **Ceiling effect** (99%+ 정확도) → 변별력 부족
- PRP: 난이도 높음 or 개인차 큼 → 노이즈 증가
- 증거: **강력** (Task difficulty 분석 확인)

측정 지표:

- Accuracy feedback sensitivity: 부정 피드백 후 정확도 변화
- Adaptive index: 정확도 개선 - RT 비용

예상 패턴:
- 남성: 부정 피드백에 과민 → RT↑ but Acc↓ (비효율적)
- 여성: 부정 피드백에 적응적 → RT↑ and Acc↑ (효율적)

V. 종합 결론
--------------------------------------------------------------------------------

WCST 특이성의 주 원인 (증거 강도 순):

1. **Task Difficulty (가장 강력)**:
   - Stroop ceiling effect (99%+) → 변별력 상실
   - WCST 적절한 난이도 → 개인차 포착 가능
   - 증거: 직접 측정, 명확한 ceiling 확인

2. **피드백 민감도 (중간 강도)**:
   - WCST만 trial-by-trial 피드백 제공
   - 외로운 사람의 사회적 신호 민감성과 일치
   - 증거: 데이터 수집 완료, 분석 진행 중

3. **불확실성 내성 (중간 강도)**:
   - WCST의 probabilistic 특성
   - 외로운 사람의 불확실성 회피 성향
   - 증거: 부분적, 샘플 크기 제한 (N=31)

4. **Proactive Control (약한 증거)**:
   - 이론적 타당성 있음
   - 증거: 제한적, 추가 분석 필요


임상적 함의:
- WCST는 외로움의 EF 영향을 측정하는 **민감한 도구**
- Ceiling 없는 task 선택의 중요성
- 피드백 제공 여부가 효과 크기에 영향
