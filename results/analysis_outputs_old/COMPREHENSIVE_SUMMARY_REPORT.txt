PART 1: 연구 개요
--------------------------------------------------------------------------------
연구 질문: UCLA 외로움이 집행기능(EF) 수행에 영향을 미치는가?
측정 과제:
  - Stroop 과제 (간섭효과)
  - WCST (위스콘신 카드 분류 검사 - 보속오류율)
  - PRP (심리적 불응기 - 병목효과)
공변인: DASS-21 (우울, 불안, 스트레스)
샘플: N=72 (여성 45명, 남성 27명)

PART 2: 주효과 분석 결과
--------------------------------------------------------------------------------

2.1 Zero-order 상관 (UCLA ↔ EF)

  Stroop 간섭: r = 0.029
  WCST 보속오류율: r = -0.100
  PRP 병목효과: r = 0.054

→ 모든 상관이 매우 약함 (|r| < 0.10)

2.2 위계적 회귀 (DASS 통제 후 UCLA 효과)

  Stroop Interference:
    ΔR² = 0.0088, p = 0.4376
  Perseverative Error Rate:
    ΔR² = 0.0024, p = 0.6871
  Prp Bottleneck:
    ΔR² = 0.0007, p = 0.8292

→ DASS 통제 후 UCLA의 고유 예측력 전무 (모든 p > 0.4)

PART 3: 성별 조절효과 ⭐⭐⭐
--------------------------------------------------------------------------------

3.1 WCST 보속오류율에서 UCLA × Gender 상호작용

  여성 slope: β = -0.3041, p = 0.7241
  남성 slope: β = 2.2902, p = 0.0675
  상호작용: β = 2.5944, p = 0.0042

  Permutation test: p = 0.0040 ***
  Bootstrap 95% CI: [0.7985, 4.4988]
  (CI가 0 제외: True)

→ 남성에서 외로움↑ → 보속오류↑ (β=2.29, p=0.068)
→ 여성에서는 관계 없음 (β=-0.30, p=0.724)
→ 상호작용 매우 유의 (p=0.004, permutation p=0.004)

3.2 성별별 상관계수

  여성:
    WCST: r = -0.249, p = 0.099
  남성:
    WCST: r = 0.241, p = 0.227

PART 4: 신뢰도 보정
--------------------------------------------------------------------------------

4.1 측정 신뢰도 (Spearman-Brown corrected)

  STROOP: r = 0.578
  WCST: r = 0.600
  PRP: r = 0.536

→ Stroop (0.58), PRP (0.54), WCST (0.60) - 모두 낮은 신뢰도

4.2 신뢰도 보정 후 상관

  STROOP: r_obs = 0.029 → r_true = 0.042
  WCST: r_obs = -0.100 → r_true = -0.140
  PRP: r_obs = 0.054 → r_true = 0.080

→ 보정 후에도 효과크기 작음 (|r| < 0.15)

4.3 검정력 (보정된 효과크기 기준)

  STROOP: N = 4523 필요 (현재 N=72)
  WCST: N = 401 필요 (현재 N=72)
  PRP: N = 1212 필요 (현재 N=72)

→ WCST 진정 효과 탐지에 N=401 필요 (power=0.80, α=0.05)
